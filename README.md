# ✨ MouseTrails

> 让每一次右键拖动，都成为一次自然的 macOS 操作。🖱️

MouseTrails 是一款原生 macOS 轨迹手势工具。按住鼠标右键或触控板的辅助点击，随手画出方向、折线或方框，就能复制粘贴、管理窗口、截图贴图、OCR，或执行你自己的动作。

它把手势留在需要它的时候，也把 macOS 原本的触控板体验还给你。🌿

## 🚀 下载

从 [最新 Release](https://github.com/Nina-17/MouseTrails/releases/latest) 下载 DMG，拖入“应用程序”文件夹后启动即可。

首次启动时，交互式教程会引导完成权限授权和几项真实操作。MouseTrails 只驻留在菜单栏；点击菜单栏图标即可打开设置。

> 公开版本目前为 ad-hoc 签名。若 macOS 阻止首次打开，请在“系统设置 → 隐私与安全性”中选择仍要打开。

## 🎨 你可以做什么

### 🌀 一笔完成日常操作

- **复制 / 粘贴 / 前进 / 后退**：用最短的上下左右手势完成高频动作。
- **窗口掌控**：居中、最小化、全屏切换、关闭窗口或退出应用。
- **空间与系统视图**：调度中心、App Exposé、显示桌面、启动台、切换空间。
- **按应用定制**：同一轨迹在不同 app 中可以拥有不同含义。
- **动作序列**：一个手势可串联多个动作。⚡️

### 📌 截图、贴图与 OCR

- 顺时针方框：直接截取框选区域，生成可拖动的置顶贴图。
- 逆时针方框：离线识别画面文字，自动复制结果并推送摘要。
- 贴图支持拖动、折叠、透明度调节、复制和另存为 PNG。
- 全部 OCR 由 macOS Vision 在本机完成，不上传截图内容。🔒

### ✍️ 这才是你的手势

除了方向、折线和方框，MouseTrails 还支持录制个人手势：连续画三次同一轨迹，便可以绑定搜索选中文字、快捷键、应用或动作序列。

轨迹颜色也可以按你的桌面风格自由调整。🌈

### 🎚️ 边缘滚动

- 屏幕左边缘滚动：调节亮度
- 屏幕右边缘滚动：调节音量

## 🖱️ 如何触发

| 设备 | 操作 |
|---|---|
| 鼠标 | 按住右键 → 拖动绘制 → 松开 |
| 触控板 | 按住辅助点击（右键）→ 拖动绘制 → 松开 |

短点击仍然是普通右键菜单。MouseTrails 不接管双指滚动、缩放、Mission Control 或 macOS 三/四指手势；它们继续按系统原样工作。✅

触控板建议使用可持续按住的辅助点击。若常用触控板，将按压力度调轻会更容易画出流畅轨迹。

## 🧭 默认手势

| 手势 | 默认动作 |
|---|---|
| 上 | 复制 |
| 下 | 粘贴 |
| 左 / 右 | 后退 / 前进 |
| 下→右折线 | 关闭窗口或标签页 |
| 右上直线 | 全屏切换 |
| 左下直线 | 最小化 |
| 下→左折线 | 关闭所有类似窗口 |
| 顺时针方框 | 生成贴图 |
| 逆时针方框 | 离线 OCR 并复制结果 |

教程还会引导你录制一个自定义搜索手势（推荐画 `S`），并保存进你自己的配置。🎓

## 🔐 权限与隐私

MouseTrails 仅在需要时请求两项 macOS 权限：

- **辅助功能**：监听右键轨迹并执行窗口、快捷键等动作。
- **屏幕录制**：仅用于你主动触发的贴图、截图与 OCR。

不上传按键、剪贴板、截图或 OCR 内容；OCR 默认离线完成。权限页可直接跳转到对应系统设置，并提供拖入授权列表的辅助窗口。

## 🔄 更新

MouseTrails 使用 Sparkle 更新机制。更新清单与 DMG 均经过 Ed25519 签名验证；你可以在“设置 → 通用”手动检查更新，也可开启自动下载，在退出并重启应用后完成安装。

## ⚙️ 配置与帮助

- 菜单栏 → **打开设置…**：调整手势、颜色、边缘滚动和权限。
- 设置 → **通用 → 使用教程**：随时重新体验交互教程。
- 菜单栏 → **打开配置文件**：查看或备份个人配置。

配置文件位于：

```text
~/Library/Application Support/MouseIncMac/config.json
```
